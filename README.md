# THE CANNON POST

## Project Overview

**The Cannon Post** is a football news website dedicated to Arsenal Football Club. It provides fans with the latest news, match analysis, transfer updates, videos, community discussions, and subscription services in a modern and responsive design.

---

## Features

* 📰 Latest Arsenal news
* 🎥 Videos section with dropdown menu
* 📅 Fixtures & Results page
* 👥 Community page
* 👤 User Sign Up and Login
* 💳 Subscription page
* ℹ️ About Us page
* 📱 Responsive navigation bar for desktop and mobile
* 🎨 Modern Arsenal-inspired black, white, and red theme

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts (Oswald & Inter)

## Installation

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Ensure all HTML, CSS, JavaScript, and image files remain in their correct folders.
4. Open **index.html** in your browser or use the VS Code **Live Server** extension.

## Design

The website uses:

* **Oswald** font for headings and branding.
* **Inter** font for body text.
* Arsenal-inspired colours:

  * Red (#e01020)
  * Black (#111111)
  * White (#FFFFFF)

The layout is fully responsive and adapts to desktop, tablet, and mobile devices.
